# Scope
This is video 84 on a Home Assistant ESPHome/LVGL embedded solution. In this video, we learn about the device and its capabilities.  We demonstrate the manufacturer's Demo, and then enhance it with a LVGL Meter widget.  We even show how we mounted the device in a 3D print that we include at our GitHub site.  The Test Rig is just the device, but its possible to add sensors through a connection at the back of the device. 

You can fetch the firmware and programs from our GitHub site, and begin using them immediately.  

In this video, 
 - Research the Knob and learn enough to program it.
 - Demonstrate the manufacturer's demo and our change to use the LVGL meter widget.
 - Review the Knob specifications and how we created the Test Rig.
 - Discuss the 3D print of the Test Rig mount (we include the source and STL files.)
 - Explain in-detail the LVGL Meter widget and its parts.
 - Review the YAML changes.

The code for this video is available at the GitHub site:
https://github.com/kwinter745321/myHA/tree/main/Videos/video84
