# Scope
This is video 91 on a Home Assistant ESPHome/LVGL embedded solution. In this video, we look back at the 2026.3 Release and its ability to handle the entire Raspberry Pi Pico family of microcontrollers.  This video walks RPI Pico users through the steps to build and deploy ESPHome.  The release is briefly discussed.  We demonstrate a LVGL 9.5 screen with LVGL Widgets using advanced styles.   

The Test Rigs is a: Raspberry Pi Pico-2W with ILI9341 Display. The wiring is provided.  You can fetch the YAML file from our GitHub site, and begin using it immediately.   

In this video, 
    • Look at the 2026.3 Release as it pertains to Raspberry Pi Pico's.
    • Demonstrate a LVGL screen with four sets of LVGL Widgets using advanced styles.
    • Review each of the four setup steps to from installing ESPHome to deploying.
    • Discuss the Test Rig and wiring.
    • Discuss a LVGL Simulator designed for ESPHome.
